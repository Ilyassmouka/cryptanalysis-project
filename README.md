# Cryptanalysis Project

This project compares two encryption algorithms:
- Caesar Cipher
- Custom Labyrinthe (polyalphabetic) Cipher

## Features
- Message generation
- Encryption using two methods
- Brute-force attack simulation
- Performance comparison
- Graph visualization using Matplotlib

## Purpose
This project was created for educational purposes to understand:
- basic cryptography
- brute-force attacks
- algorithm complexity

## Technologies
- Python
- Matplotlib
